# Amazon Clone

A responsive front-end clone of the Amazon homepage built using **HTML5** and **CSS3**. This project recreates the visual design and layout of Amazon's website while focusing on responsive web design, structured HTML, and modern CSS styling.

## Features

* Amazon-inspired navigation bar
* Search bar with category selection
* Location and account sections
* Shopping cart interface
* Product category cards
* Hero banner section
* Responsive page layout
* Amazon-style footer
* Clean and organized UI

## Technologies Used

* **HTML5** — Used to create the structure and content of the webpage
* **CSS3** — Used for styling, layout, responsiveness, and visual design

## Getting Started

To run this project locally:

1. Clone the repository:

```bash
git clone YOUR_REPOSITORY_URL
```

2. Open the project folder:

```bash
cd Amazon-Clone
```

3. Open the `index.html` file in your browser.

You can also use the **Live Server** extension in Visual Studio Code to run the project.
